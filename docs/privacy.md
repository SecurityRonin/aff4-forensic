# Privacy Policy

*Last updated: 2026-06-30*

## Summary

aff4-forensic is a local Rust library. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

aff4-forensic reads only the AFF4 container bytes you pass to it. All processing happens in memory on your local machine. Nothing is uploaded anywhere.

## Telemetry

aff4-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

aff4-forensic is open source. You can audit every line of code at [github.com/SecurityRonin/aff4-forensic](https://github.com/SecurityRonin/aff4-forensic).
